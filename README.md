# Green-Ledger
Still writing your expenses on paper and losing track later? Install and try this app now.
Track all your cash and online money in one place with a simple, manual system. Manage personal ledgers, log your trading performance, and stay in control of your finances.

Includes PIN protection and data export for secure backup and easy transfer anytime.

🚀 Features

* 💰 Track expenses across multiple accounts (Cash & Online)
* 📊 Monitor trading performance (profit/loss, charges, net, % returns)
* 📒 Personal ledger tracking (money given/taken)
* 🔐 Secure PIN protection for privacy
* 💾 Export data for backup and easy transfer between devices
* 🎯 Clean dark UI with black + neon green theme
* ⚡ Fast and simple manual entry system
